# Examples from my GM notes
A session isn't an event, but it's contextual info. One session might contain many events
in my notes so far, one in-game day mostly equals 1 event since I didn't have the tools to do better.
Also, I originally just tracked how many days had passed in game, not the actual date, just ignore any "day X" notes.
I've marked out the contents of 1 event. These aren't in the expected format, it's just an example of the kind of content you'll get when we start exporting events from my existing notes.
